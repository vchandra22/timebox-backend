# Architecture Decision Records

ADR mencatat keputusan arsitektur yang berdampak ke struktur project,
dependency direction, naming, atau standar kerja tim.

## Format

Gunakan format berikut untuk ADR baru:

```md
# 0000. Short Decision Title

## Status

Proposed | Accepted | Deprecated | Superseded

## Context

Masalah atau tradeoff yang membuat keputusan ini perlu dicatat.

## Decision

Keputusan yang dipilih.

## Consequences

Dampak positif, negatif, dan batasan dari keputusan.
```
